
Wordpress Snippets
==========================================

使い方
-------------------------------------------------------------------------------------------

### インストール
1. アーカイブを展開する
2. プロジェクトディレクトリのbundlesディレクトリにwordpress.rubleをコピー

### 動作確認
1. ファイルを作成する
2. get_headerとタイプする
3. Tabキーを押す
4. スニペットが挿入されれば動いています。
