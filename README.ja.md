
Aptana Snippets
================================

About
---------------------------------------------------------------------

このスニペット集はAptana用のスニペットです。
スニペットをカテゴリ別に分類してあります。

カテゴリは下記の通りです。

* Zend Freamwork
  *directory:* zend-framework

* Aptana snippet template
  *directory:* template

* Google Sitemap Template
  *directory:* sitemap

* OOCSS CSS Framework
  *directory:* oocss

*Javascript Framework Mootools
  *directory:* mootools

*Javascript Framework jQuery
  *directory:* jquery

* Google Maps
  *directory:* google-maps

* DooPHP Fremework
  *directory:* doophp

* Django Fremework
  *directory:* django

* CSS Code
  *directory:* css

* Concrete5 CMS
  *directory:* concrete5

How to use
---------------------------------------------------------------------

スニペット集はプロジェクトフォルダにコピーするだけで利用できるようになります。

### Step 1. Aptanaプロジェクトの作成
メニューより新しいプロジェクトを作成します。

### Step 2. snippetsディレクトリの作成
プロジェクトディレクトリの配下にsnippetsディレクトリを作成します。

### Step 3. アーカイブの展開
ダウンロードしたアーカイブを展開します。

### Step 4. 必要なスニペットのコピー
展開したスニペットの中から必要なものをコピーし、snippetsディレクトリにコピーします。

### Step 5. スニペットビューの表示
ウィンドウ、ビューの表示からスニペットを選択します。
コピーしたスニペットが表示されれば完了です。
